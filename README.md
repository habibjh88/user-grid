A lightweight plugin for display WordPress users in Gutenberg Editor

## Description ##

This plugins will help to display your users for gutenberg editor. 

## Developement Guide ##
**Install:**
* npm install -f
* composer install

**Start and Watch CSS/JS**
* npm start 
* npm run watch

**Package Create**
* npm run package
* npm run zip
